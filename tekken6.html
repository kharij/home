Installation

  • Extract the downloaded .zip/.7z file with a file archiver program. ex. 7-zip, WinRar
  • Navigate to TEXTURES folder in the PPSSPP directory:
    • For Windows .zip versions this is located in ../memstick/PSP/TEXTURES
    • For Windows installer versions this is located in %UserProfile%/Documents/PPSSPP/PSP/TEXTURES
  • Copy the ULUS10466 folder into the TEXTURES folder, and you're done.
